# 二叉树

## 遍历方式

### 前序遍历
递归解法：

```
 public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        dfs(root, res);
        return res;
    }
    
    private void dfs(TreeNode root, List<Integer> res) {
        if (root == null) {
            return;
        }
        // 根据前序、中序、后序改变顺序。
        res.add(root.val);
        if (root.left != null) {
            res.add(root.left.val);
        }
        if (root.right != null) {
            res.add(root.right.val);
        }
    }
```
