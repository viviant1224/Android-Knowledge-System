# 创建型设计模式

## 单例模式

### 饱汉模式（懒汉模式）
```
public class SingleSton {
    private static SingleSton sIntance = null;
    private SingleSton() {
    }
    
    public static getInstance() {
        if (sIntance == null) {
            sIntance = new SingleSton();
         }
        return sIntance;
    }
}
```

优点：懒加载启动快，资源占用小，使用时才实例化，无锁。
缺点：非线程安全。
